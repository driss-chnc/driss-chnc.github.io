# ABI – Site e-commerce de vêtements

Projet de site vitrine pour une boutique de vêtements appelée **ABI**.
Le site présente une page d’accueil avec une bannière, des produits populaires et une section vêtements.

## Aperçu du projet

Le site contient :

* Une **page d’accueil avec une bannière (hero section)**
* Une section **Nos meilleures ventes**
* Une section **Nos habits**
* Une **navigation simple**
* Un design moderne avec images et cartes produits

## Technologies utilisées

* **HTML5**
* **CSS3**
* **JavaScript** 
* **GitHub Pages** pour l’hébergement

## Installation

1. Cloner le projet :

```
git clone https://github.com/driss-chnc/driss-chnc.github.io
```

2. Ouvrir le dossier.

3. Lancer le fichier :

```
index.html
```

dans un navigateur.
